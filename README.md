# simpleujs

**simpleujs** is a small library to perform unobtrusive javascript operations using `data-*`
attributes.


## Status

> **This project is still experimental, use with caution!**


## Installation

Using NPM:

```bash
$ npm add simpleujs
```

Using Yarn:

```bash
$ yarn add simpleujs
```
